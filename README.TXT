Projek ini merupakan pengembangan aplikasi game Pac-Man menggunakan bahasa pemrograman Java dan framework Greenfoot Aplikasi ini merupakan versi klasik Pac-Man yang sudah dimodifikasi untuk menambahkan tantangan baru dalam permainan. Fungsinya meliputi:

Kontrol karakter Pac-Man termasuk pergerakan yang responsif terhadap input dari pengguna.
Algoritma pengejaran hantu yang dinamis, di mana setiap hantu memiliki pola gerakan yang berbeda-beda untuk mengejar Pac-Man.
Pengumpulan poin saat Pac-Man memakan titik-titik yang tersebar di peta permainan.
Interaksi objek seperti power-ups yang memberi kemampuan khusus kepada Pac-Man untuk sementara waktu.

Game ini dapat dijalankan di **platform desktop** dan bertujuan untuk memberikan pengalaman bermain yang interaktif dan seru, baik untuk pengguna yang ingin bernostalgia maupun pemain baru yang menyukai permainan arkade klasik.
